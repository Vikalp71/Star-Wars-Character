# Star-Wars-Character
A responsive React app that fetches and displays Star Wars characters using the SWAPI API. Includes character details, pagination, dynamic images, and modal views — built with React, JavaScript, and CSS.

--Tech Stack --
  HTML5
- CSS3
- JavaScript (ES6+)
- React (hooks, functional components)
- SWAPI (https://swapi.dev/)

--Features--

1- Fetches Star Wars characters from the SWAPI API

2- Displays character cards with random images

3- Pagination for browsing through all characters

4- Click any card to view detailed information

5- Handles loading and error states gracefully

6- Fully responsive (mobile, tablet, and desktop)

7- Clean UI built using pure CSS
